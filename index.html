Simulando uma Epidemia

Bem vindo ao Simulando uma epidemia! Esse projeto é uma tentativa de demonstrar visualmente o desenvolvimimento de uma epidemia. Porém, é importante notar que essa é somente uma ferramenta didática, o objetivo desse projeto não é demonstrar com precisão uma epidemia, mas reforçar a importância do distanciamento e de outras medidas preventivas.
